Portfólio

Estes são alguns dos projetos que desenvolvi durante meus estudos.

  • MEMORY-GAME

    • Este projeto consiste em um jogo da memória simples e interativo, desenvolvido inteiramente com HTML5, CSS3 e JavaScript puro, sem o uso de bibliotecas ou frameworks. O objetivo do jogo é encontrar os pares de cartas iguais no menor tempo e com o menor número de jogadas possível.

    Tecnologias utilizadas:

    • HTML: estrutura das cartas e da interface;
    • CSS: estilo visual moderno e responsivo;
    • JavaScript: lógica do jogo (embaralhamento, controle de clique, comparação de cartas, contagem de jogadas).
  • Weather-Forecast

    • Este projeto é um site de consulta meteorológica onde o usuário digita o nome de uma cidade e recebe, em tempo real, as informações climáticas atuais daquele local. Os dados são obtidos através de uma API pública de clima (como a OpenWeatherMap).

    Tecnologias utilizadas:

    • HTML: estrutura básica da página e do formulário de pesquisa;
    • CSS: layout responsivo com design moderno e limpo;
    • JavaScript:integração com a API, tratamento de respostas, e atualização dinâmica dos dados na interface.
